
# App Con react + axios + tailwind

Aplicacion basica de react consumiendo la api de PokeApi usando la libreria de axios y tailwind



## Instalacion

Para instalar las depencias, ejecute el siguiente comando en su terminal

```bash
  npm install
```

Iniciar el servidor

```bash
  npm run dev
```
